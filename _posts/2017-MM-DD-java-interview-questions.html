Java platform: - JVM - API

Difference between C++ and Java? - Pointers - Garbage collection - Multiple inheritance (But has Interfaces and Polymorphism) - Destructors (But has finalize() method and garbage collection) - Structures or unions (Java Collections Framework) - Global variables -Global functions

Static initializer code block? - called when a reference to a static field is created/used anywhere in the app - code block with no keyword before the opening brace. - - - - - - - -

Instance initializer code block? - When object is instanciated by calling constructor, the instance code block in the class definition is executed before the constructor method is executed - why not include it in constructor? cz if we have multiple constructors that share some code then we can put that code outside the constructors & inside code block and it will still be executed no matter which constructor is called to instantiate the object. - - - - - - - -
